# ProjectHIM
Unity / C# / Pun / Spring / Mysql 을 연동해 만든 온라인 게임

# Game Info
플레이어는 중세 몬스터 종족을 선택 해 플레이 하며
함께 마을을 약탈 할 "작업반" 인원들을 모아 시나리오에 진입 해
마을의 시민을 살해하고 물건을 약탈 하여
보스NPC를 처치하고 탈출구가 열리면 시간안에 탈출하는
PVPVE 형태의 타르코프 라이크 게임입니다
